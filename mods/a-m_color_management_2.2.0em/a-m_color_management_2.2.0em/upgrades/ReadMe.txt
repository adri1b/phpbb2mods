Admins/mods color management MOD - upgrade instructions
-------------------------------------------------------

In order to use one of these upgrade instructions with EasyMOD:

#1 copy the concerned file from this directory to the MOD root directory
#2 delete the MOD instructions file "a-m_color_management_x.x.xem.txt"
#3 copy the MOD directory to your phpBB admin/mods directory.

At last, you can apply the new colors for some other
MODs features if you have these MODs already installed:
- Fading links >= 1.0.2
- Global announcement >= 1.2.8
- Last visit >= 1.2.5
- Top X posters >= 1.0.1

The instructions/MODs are in the /optional_features/ directory.
