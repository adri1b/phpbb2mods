BBCodes & smilies enhancement MOD - upgrade instructions
---------------------------------------------------

In order to use one of these upgrade instructions with EasyMOD:

#1 copy the concerned file from this directory to the MOD root directory
#2 delete the MOD instructions file "BBCodes-smilies_enhancement_x.x.xem.txt"
#3 copy the MOD directory to your phpBB admin/mods directory.
