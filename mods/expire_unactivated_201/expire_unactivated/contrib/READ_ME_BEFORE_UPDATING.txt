Expire Unactivated Accounts By Timeframe Update instructions.

If you currently have versions 1.1.0 or 1.2.0 installed you may use the included updater mod file 1_2_to_2_0_updater.txt in the Contrib folder to update your
version of the mod to 2.0.1. Besides running the mod updater file in Easymod there are no DIY requirements. 

If you currently have version 1.0.0 installed you may use the included legacy updater mod file in the Contrib folder to update your
version of the mod to 1.2.0., you may then run the 1_2_to_2_0_updater.txt mod to get to the full 2.0.1 release.
