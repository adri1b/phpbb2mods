There are two versions of this MOD. One that by clicking on icon will inwoke web page with web based Lite version of Yahoo! Messanger. This is what original icon was doing. If you wish to istall that version use this file:

yahoo_online_offline_204_web.mod

Second version will invoke PC version of Yahoo messanger that is (or need to be) installed on your PC. Use this file for installing this verion:

yahoo_online_offline_204_pc.mod
