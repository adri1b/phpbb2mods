ACP User Registration Update Scripts

The scripts found in the updates/ directory will allow you to update from a previous version of ACP User Registration. These scripts can be run manually or via EasyMOD. Find the version of ACP User Registration you are currently using below to determine which script to run.

Version 1.0.1 -- Run updatefrom_101.txt
Version 1.0.0 -- Run updatefrom_100.txt
Version 0.1.1 (Release Candidate) -- Run updatefrom_011.txt
Version 0.1.0 (beta) -- Run updatefrom_010.txt
