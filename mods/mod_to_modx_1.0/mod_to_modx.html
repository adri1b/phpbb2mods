mod_to_modx Convertor

System Requirements

Linux

  • Any Linux distribution running MONO; which can be downloaded from http://www.mono-project.com/Downloads
  • 32MB RAM
  • 200MHz processor, faster recommended
  • 5MB HDD space

MAC OSX

  • MONO; which can be downloaded from http://www.mono-project.com/Downloads
  • 32MB RAM
  • 200MHz processor, faster recommended
  • 5MB HDD space

Windows

  • Windows 98, NT4, 2000, XP or Vista
  • The .net framework 1.1 or MONO 1.1
  • 32MB RAM
  • 200MHz processor, faster recommended
  • 5MB HDD space

Using mod_to_modx

Linux, MAC OSX and MONO on Windows®

  1. Make sure you have the minimum system requirements
  2. If you haven’t ready, download mod_to_modx from phpBB.com
  3. Extract mod_to_modx.zip if you haven’t already
  4. Start a terminal window/bash/command window
  5. Type the following (using the path mod_to_modx.exe is located, here it is /root/release/)
[root@fedora ~]# mono /root/release/mod_to_modx.exe
  1. The following will appear

The phpBB MOD Team MOD Template to MODX conversion script.
Please enter the filename of a file to convert:
  1. Type in the name of the file to convert, for example
/root/release/sql_backup/sql_backup_v1.0.2/sql_backup_v1.0.2.mod
  1. Press Enter.
  2. The file will be converted and the following message will be displayed
Conversion is complete, file saved as

[root@fedora ~]#
  1. All done, a file named modx.subsilver.en.xsl will also have been copied, open the converted XML file in Firefox1 to check that it converted successfully.

The .net Framework on Windows® (best for windows)

  1. Make sure you have the minimum system requirements
  2. If you haven’t ready, download mod_to_modx from phpBB.com
  3. Extract mod_to_modx.zip if you haven’t already
  4. You now have several options
    1. Drag the file you wish to convert onto the executable file (mod_to_modx.exe) and the file will be converted automatically. A file named modx.subsilver.en.xsl will also have been copied. Open the converted XML file in Internet Explorer or Firefox to check that it converted successfully.
    2. Open a command prompt by going, start » run » type "cmd" » Enter.

      1. Type the following (using the path mod_to_modx.exe is located, here it is c:\mod_to_modx\)

        C:\>c:\mod_to_modx\mod_to_modx.exe

      2. The following will appear:

      3. Type in the name of the file to convert, for example

        c:\downloads\sql_backup\sql_backup_v1.0.2\sql_backup_v1.0.2.mod
      4. Press Enter.

      5. Alternatively type the following (all on one line) and Press Enter:
