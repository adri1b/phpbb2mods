Quick Search Update Scripts:

The scripts found in the updates/ directory will allow you to update from a previous version of Quick Search. These scripts can be run manually or via EasyMOD. Find the version of Quick Search you are currently using below to determine which script to run.

Version 3.0.1 -- Current Version
Version 3.0.0 -- Install upgradefrom_300.txt
Version 2.1.1 -- Install upgradefrom_211.txt
Version 2.1.0 -- Install upgradefrom_210.txt
Version 2.0.0a -- Install upgradefrom_200.txt
Versions previous to 2.0.0a -- No upgrade script is available for your version. Uninstall your copy of Advanced Version Check and run a fresh install of this version.
