Registration Stopper Update Scripts:

The scripts found in the contrib/updates/ directory will allow you to update from a previous version of Registration Stopper. These scripts can be run manually or via EasyMOD. Find the version of Registration Stopper that you are currently using below to determine which script to run.
Please note that if you intend to use any of these scripts via EasyMOD, you will need to move them into the root directory of this zip for them to install properly.

Version 1.2.0 -- Current Version
Version 1.0.1 -- Install upgradefrom_020_101.txt
Version 1.0.0 -- Install upgradefrom_020_101.txt
Version 0.2.0 -- Install upgradefrom_020_101.txt
Version 0.1.0 -- Install upgradefrom_010.txt

The following update scripts are also available in MODX Template format. At the time of writing, these templates are not yet compatible with EasyMOD, but can be installed manually. Find the version of Registration Stopper you are currently using below to determine which script to run.
Please note that if you intend to use any of these scripts, you will need to move them into the root directory of this zip for the XSL to open correctly.

Version 1.2.0 -- Current Version
Version 1.0.1 -- Install upgradefrom_020_101.xml
Version 1.0.0 -- Install upgradefrom_020_101.xml
Version 0.2.0 -- Install upgradefrom_020_101.xml
Versions previous to 0.2.0 -- No MODX update script is available for your version; please use one of the provided scripts in Text Template format (see above).
